## 二、Part One：HTML & CSS 学习轨迹

### 阶段一：HTML 基础 + CSS 入门（Day01-Day04）

| 日期 | 主题 | 关键知识点 |
|------|------|-----------|
| Day01 | HTML 初识 | HTML文档结构、纵向色块搭建（Flex布局雏形） |
| Day02 | CSS 引入方式 | 内联/内部/外部样式、基础选择器、色块布局 |
| Day03 | 盒子模型 | `width/height`、`border`（三角形技巧）、`border-radius`、`padding`、`margin`、外边距抢占问题、`float`浮动、聊天框实战 |
| Day04 | 横向布局 + 表单 | 横向色块搭建、HTML常见标签（h1-h6/p/a/img等）、表单标签（input/button/select） |

**阶段特征**：从零起步，掌握 HTML 基本骨架和 CSS 盒子模型核心。通过"色块搭建"练习建立空间布局意识，三角形成型技法初现设计思维。

### 阶段二：CSS 进阶 + 定位体系（Day05-Day07）

| 日期 | 主题 | 关键知识点 |
|------|------|-----------|
| Day05 | CSS 属性深化 | 文本操作、列表样式、高级选择器、表单样式、背景属性（`background-size/position/repeat`） |
| Day06 | 定位系统 | 元素类型(`inline/block/inline-block`)、垂直对齐、**固定定位**（`fixed`）、**粘性定位**（`sticky`）、**相对定位**（`relative`）、**绝对定位**（`absolute`） |
| Day07 | 补充 + 自适应 | 锚点定位、透明色、表格、溢出属性（`overflow`）、宽高自适应、`calc()`计算属性、上下固定布局 |

**阶段特征**：CSS 定位体系是重点突破阶段，完整覆盖5种 `position` 值。自适应布局思维开始形成，"上下固定+中间自适应"的经典布局模式在此建立。

### 阶段三：综合实战 + HTML5/CSS3（Day10-Day13）

| 日期 | 主题 | 关键知识点 |
|------|------|-----------|

| Day11 | 布局进阶 | 滑块效果、单行省略号、浮动高度自适应、**两栏布局**、**三栏布局**、上下固定布局、元素隐藏方式 |
| Day12 | HTML5 + CSS3选择器 | HTML5语义化标签（header/main/nav/footer）、视频/音频标签、表单补充、层级选择器、属性选择器、结构伪类选择器（`:nth-child`等） |

**阶段特征**：从"知识点"过渡到"综合页面搭建"，首次完整复刻商业页面（小米官网）。HTML5 语义化 + CSS3 新特性吸收完整。

| 日期 | 主题 | 关键知识点 |
|------|------|-----------|

**阶段特征**：编程思维建立期——从变量声明到循环嵌套，通过"打印三角形/菱形"等算法题培养逻辑思维。AI辅助方案对比意识萌芽。

| 日期 | 主题 | 关键知识点 |

## 六、技术领域及知识点汇总

### HTML
- 语义化标签（header/main/nav/footer/section/article）
- 表单元素全系列（input/select/textarea/button/label）
- 多媒体标签（video/audio）
- 锚点定位、SEO基础

### CSS
- **布局体系**：浮动(float)、定位(position 5种)、Flexbox(弹性盒子)、多列(column)
- **盒子模型**：标准盒模型、怪异盒模型(border-box)、margin塌陷
- **响应式**：媒体查询(@media)、vw/vh视口单位、rem适配
- **动画系统**：过渡(transition)、2D变换、3D变换(透视+立方体)、关键帧动画(@keyframes)、逐帧动画(steps)
- **CSS3选择器**：属性选择器、结构伪类(:nth-child/:first-child)、目标伪类(:target)、否定伪类(:not)
- **视觉增强**：阴影(box-shadow/text-shadow)、渐变、自定义字体(@font-face)、矢量图标(iconfont)、CSS Sprite、混合模式(mix-blend-mode)

### JavaScript
- **基础语法**：变量(var/let/const)、数据类型、类型转换、运算符、条件语句、循环
- **函数**：声明/表达式/箭头函数、参数/返回值、递归、闭包、柯里化、预解析(hoisting)
- **内置对象**：Math/Date/String/Array/Object方法全系
- **DOM**：节点查询/创建/删除/克隆、样式操作、属性操作、类名操作
- **事件系统**：事件绑定/解绑、事件对象、事件传播(捕获/冒泡)、事件委托、键盘/表单/触摸事件
- **BOM**：window对象、浏览器尺寸、滚动控制、location/history、弹出层
- **ES6+**：箭头函数、解构赋值、展开运算符、模板字符串、Class类
- **高级概念**：this绑定规则、call/apply/bind、闭包、函数柯里化、防抖(debounce)、正则表达式、发布订阅模式
- **Canvas API**：2D渲染、游戏循环(requestAnimationFrame)、碰撞检测(AABB)、粒子系统
- **异步基础**：setTimeout/setInterval、clearTimeout

### 工程化
- **Webpack 5**：模块打包、开发服务器、生产构建
